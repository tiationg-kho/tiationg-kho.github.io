## Stack vs Heap

|            | Stack                                               | Heap                                    |
| ---------- | --------------------------------------------------- | --------------------------------------- |
| Store      | Method calls, primitive types, reference to objects | Objects                                 |
| Management | Automatic                                           | Manual or Automatic (Garbage Collector) |
